# Changelog

All notable changes to the Voice Form project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-04-29

### Added
- Initial release of the Voice Form application
- Voice input functionality using Web Speech API
- Modern UI with responsive design
- Real-time visual feedback during voice recording
- Form validation for all fields
- Cross-browser support (best in Chrome)
- GitHub Pages deployment configuration
- Automated deployment with GitHub Actions

### Technical Details
- Built with React 19.1.0
- CSS variables for consistent theming
- Enhanced accessibility features
